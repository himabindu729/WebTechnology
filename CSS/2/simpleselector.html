Group selector

Universal selector is represented with *(body),where it is applied to whole page.

tagname selector is represented with specified tagname of html document.

Id selector is represented with # and "id" and value should be given for ecah tag in html doc.value should be unique

Class selector is represented with .classname,it is used when more than one css property have same class

Group selector is the combination of tagname, id , class selectors.First preference is tagname.
